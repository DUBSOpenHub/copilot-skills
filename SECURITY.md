# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Please do NOT open a public issue.**

Instead, please email [security@dubsopenhub.com](mailto:security@dubsopenhub.com) or use [GitHub's private vulnerability reporting](https://github.com/DUBSOpenHub/copilot-skills/security/advisories/new).

We will acknowledge your report within 48 hours and aim to provide a fix or mitigation within 7 days.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | ✅        |

## Security Best Practices

- Keep dependencies up to date
- Never commit secrets or credentials
- Review security advisories regularly
