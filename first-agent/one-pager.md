# Your First Agent: AI Skilling for Everyone

## One-Pager for Leadership

---

### What Is It?

A live, instructor-led training program where every employee builds a working AI agent — regardless of technical background. Three sessions, ~2 hours total, zero coding required.

Powered by the GitHub Copilot CLI, the program uses a guided skill that walks each attendee through running, modifying, and building an AI agent in their terminal.

### Why It Matters

AI skilling programs typically teach theory. Employees complete training, check a box, and never use the tool again. This program flips that model: the training IS the output. Every attendee walks out with a working agent they built themselves — one that saves them time on a real task they do every week.

The goal isn't AI literacy. The goal is AI capability. Every employee becomes an agent builder.

### The Three Sessions

| Session | Name | Duration | What Happens | What They Walk Away With |
|---------|------|----------|--------------|--------------------------|
| 1 | **Run It** | 30 min | Run a pre-built agent on real meeting notes. See structured output instantly. | "AI agents are just text files. 20 lines." |
| 2 | **Remix It** | 30 min | Open the agent file. Change a rule. See behavior change. Break it, fix it. | "I can read, modify, and control an agent." |
| 3 | **Build It** | 45 min | Build a custom agent from scratch. Test it. Demo it to the room. | A working agent + a showcase card to share. |

### What Makes This Different

**No slides.** The terminal is the classroom. Attendees learn by doing, not watching.

**No coding.** Agents are written in plain English. If you can write clear instructions, you can build an agent.

**No theory.** There is no "intro to AI" module. The first thing attendees do is run a working agent. Understanding comes from experience, not explanation.

**Live energy.** The program is designed for a room, not a screen. The shared "aha" moment when 20 people realize they just built an agent — that's the moment that spreads.

**The showcase.** Session 3 ends with a live demo: 4-5 attendees show the room what they built in 60 seconds each. The agents solve real problems. The room applauds. This moment generates the demand for the next cohort.

### Sample Agents Built by Non-Developers

| Role | Agent | What It Does | Time Saved |
|------|-------|-------------|------------|
| Product Manager | "Sprint Summarizer" | Turns 50 Jira updates into a 1-paragraph status | ~15 min/week |
| HR Business Partner | "Job Description Writer" | Drafts JD from 5 bullet points | ~45 min/posting |
| Sales Rep | "Pre-Call Briefer" | Compiles prospect info into a 1-page prep doc | ~20 min/call |
| Finance Analyst | "Report Formatter" | Turns raw data notes into a formatted weekly report | ~25 min/week |
| Executive Assistant | "Meeting Summarizer" | Turns messy notes into action items + decisions | ~10 min/meeting |

### How to Run It

**Prerequisites for attendees:**
- GitHub Copilot CLI installed (IT can pre-provision)
- A laptop with terminal access
- That's it

**Prerequisites for facilitators:**
- The facilitator guide (included with the program)
- A room with a projector
- 2 hours of calendar time

**Scheduling options:**
| Format | Layout | Best For |
|--------|--------|----------|
| Sprint (recommended) | All 3 sessions in one half-day | AI Days, offsites, kickoffs |
| Weekly | 1 session per week, 3 weeks | Ongoing enablement programs |
| Bootcamp | Sessions 1+2 combined, then Session 3 | Time-constrained teams |

### How It Scales

The program is designed to scale through champions, not central teams:

1. **Pilot**: Run with one team (15-25 people). Collect showcase cards.
2. **Amplify**: Share the showcase in Slack/Teams. Demand generates itself.
3. **Champion network**: Trained attendees become facilitators for the next cohort. The facilitator guide makes this turnkey.
4. **Viral loop**: Every cohort's showcase creates sign-ups for the next one.

### Measuring Success

| Metric | Target | How |
|--------|--------|-----|
| Completion rate | >90% | Tracked by skill (SQL) |
| First agent built | 100% of attendees | Tracked by skill |
| Time to first agent | <25 minutes | Tracked by skill |
| Post-training agent usage | >50% use agent weekly | Follow-up survey at 2 weeks |
| Net Promoter Score | >70 | Post-session survey |
| Champion conversion | >10% become facilitators | Facilitator sign-up rate |

After the first cohort, expect:
- 15-25 working agents built by non-developers
- Showcase cards shared in Slack unprompted
- 2-3 people asking "When's the next session?"
- At least one agent that surprises you with its creativity

That's the program working as designed. The agents are the proof. The excitement is the marketing.
